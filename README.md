# Sales Analytics System

This project is a Python-based Sales Analytics System developed as part of the BITSOM / Masai Sales Analytics assignment.

The application reads raw sales data, validates and processes transactions, integrates product data via API simulation, enriches sales records, and generates a structured sales report.
